┌─────────────────────────────────────────────────────────────────────────────┐
│ 🥇 CÁCH 1: GITHUB DEPLOY (KHUYẾN NGHỊ - DỄ NHẤT)                           │
└─────────────────────────────────────────────────────────────────────────────┘

  Bước 1: Commit và Push
  ─────────────────────────────────────────────────────────────────────────
  git add AI-service/
  git commit -m "Add Railway deployment configuration"
  git push origin main

  Bước 2: Deploy trên Railway Dashboard
  ─────────────────────────────────────────────────────────────────────────
  1. Vào: https://railway.app/new
  2. Click: "Deploy from GitHub repo"
  3. Chọn repository: BTL-Mobile
  4. Root Directory: AI-service
  5. Click: "Deploy Now"

  Bước 3: Set Environment Variables
  ─────────────────────────────────────────────────────────────────────────
  Settings → Variables → Add:
    GOOGLE_API_KEY=your_actual_google_api_key
    GEMINI_MODEL=gemini-2.0-flash
    LLM_PROVIDER=gemini
    LLM_TEMPERATURE=0.7
    LLM_TOP_P=0.9
    LLM_MAX_TOKENS=2048

  Bước 4: Generate Domain
  ─────────────────────────────────────────────────────────────────────────
  Settings → Networking → Generate Domain
  Copy: https://your-service.up.railway.app

  ✅ XONG! Service đã live.
